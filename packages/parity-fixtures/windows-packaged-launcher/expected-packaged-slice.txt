This packaged tree is the scoped Windows packaging-visible launcher slice for
the currently verified Rust-backed help/version/config/export/transform
surface.

The artifact is intentionally a maintainer-inspectable launcher tree, not an
MSI, zip release archive, signed artifact, installer, or release-channel build.
It proves the Windows-facing package layout and startup naming for the current
development host, not a Windows-native or cross-compiled release binary.

Included:
- Slic3r-console.exe: direct Rust console runtime executable for the verified
  slice
- share/slic3r/packaged-slice.txt: this scope note

Smoke coverage proves:
- package layout
- direct executable startup handoff
- --help
- --version
- representative config persistence
- representative export and transform startup flows

Deferred:
- MSI and other installer formats
- zip release archives
- signing and release-channel automation
- Slic3r.exe GUI packaging behavior
- Windows-native or cross-compiled release binaries
- broad bundled DLL/dependency layout
- new CLI behavior beyond the verified Rust-backed slice
- shared cross-platform packaged parity evidence
